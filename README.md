# AnimateLogin

[Animated Login Example](https://drive.google.com/file/d/1GyoqVVT32r4VZLLTY3NCDSvtdoGmFeHv/view?usp=sharing)

## Setup

Clone the project to your computer.

```bash
cd AnimatedLogin
expo install
expo start
```
